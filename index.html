Política de Privacidade – Argoplan

A Argoplan utiliza a API oficial do WhatsApp Business (Cloud API) exclusivamente para comunicação com clientes que iniciam contato ou autorizam o recebimento de mensagens.

Dados coletados

  • Número de telefone
  • Conteúdo das mensagens trocadas

Finalidade

  • Atendimento
  • Comunicação institucional
  • Informações solicitadas pelo próprio usuário

Os dados não são vendidos, compartilhados ou utilizados para finalidades diferentes das descritas.
